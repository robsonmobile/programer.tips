Opening given folder in the MacOS Terminal

  • macos
14 September 2014
Improve it on Github

No matter who you are, a developer, database administrator or system engineer, Terminal might be one of the tools you've been working with most.

For example, to the front-end developers, we usually need to run the building GruntJS or GulpJS scripts, start an ExpressJS application right from the Terminal.

These tasks have a common starting point which is to be executed from given directory.

Today, I will show you two quick ways to open particular directory in the MacOS Terminal without using any additional software, scripts or workflow.

Adding the New Termnial at Folder menu

  • Open the System Preferences
  • Click the Keyboard, and choose the Shortcuts tab
  • Choose Services from the left, and then enable the New Terminal at Folder option from the right

You also can click add shortcut button to define the shortcut.

From now on, in order to open a folder in Terminal, just right-click the folder, and choose Service → New Terminal at Folder from the context menu.

Dragging the folder into the Terminal

This way is more convenient than the first one. Just hold down the command key and drag the folder into the Terminal. That's all. How easy it is!

By doing this, you don't even to worry if the path to folder consists of spaces. The system automatically add \ characters to quote the path for you.

For instance, dragging /Library/Application Support/App Store into the Terminal will execute the command:

$ cd /Library/Application\ Support/App\ Store

What about Windows?

On Windows, the job can be done easily by holding down the Shift key, right-click the folder, and choose Open command window here from the context menu.
